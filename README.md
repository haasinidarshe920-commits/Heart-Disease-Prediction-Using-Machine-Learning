# Heart-Disease-Prediction-Using-Machine-Learning

## Project Overview

This project predicts whether a patient is likely to have heart disease based on medical attributes using Machine Learning.

The model is trained using the Heart Disease UCI Dataset and Logistic Regression algorithm.

## Objective

To predict the presence of heart disease using patient medical information and Machine Learning techniques.

## Dataset

Source: Kaggle – Heart Disease UCI Dataset

Features:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Maximum Heart Rate
* Exercise Induced Angina
* ST Depression
* Number of Major Vessels

Target:

* 0 = No Heart Disease
* 1 = Heart Disease

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Feature Scaling
5. Train-Test Split
6. Logistic Regression Model Training
7. Model Evaluation

## Evaluation Metrics

* Accuracy: 79.89%
* Precision: 85.29%
* Recall: 79.82%

## Visualizations

* Confusion Matrix
* ROC Curve

## Outcome

The model successfully predicts whether a patient is likely to have heart disease and demonstrates the application of Machine Learning in healthcare analytics.
